Navigation

There are three kind of menu style in this template, Default Menu, Transparent Menu and Full Screen Menu.  You need to change some classes to switch menu styles. It's very easy, here is some example of those menus:

The markup of Default menu: 

<!-- Navigation start -->
<nav class="navbar navbar-default navbar-fixed navbar-transparent white bootsnav on no-full">
    your menu goes here...
</nav>
<!-- Navigation end -->

Just put "tt-default-nav" class with "navbar-custom" for Default Menu.


The markup of Side menu: 

<!-- Navigation start -->
<nav class="navbar navbar-default navbar-sidebar navbar-scrollspy bootsnav no-full" data-minus-value-desktop="0" data-minus-value-mobile="0" data-speed="1000">
  your menu goes here...
</nav>
<!-- Navigation end -->

You need to add "navbar-fixed-top" and  "transparent" classes with "navbar-custom" class for get a transparent fixed menu. And  also add "white" class for adjust navbar background color and menu item color.

You may add two alter logo for adjust logo color and navbar background color(applicable for transparent menu).
Here is the sample: 

<div class="navbar-header">
        <button type="button" class="navbar-toggle" data-toggle="collapse" data-target="#navbar-menu"> <i class="fa fa-bars"></i> </button>
        <div class="logo"> <a href="index.html"> <img class="logo logo-display" src="assets/images/logo-white.png" alt=""> <img class="logo logo-scrolled" src="assets/images/logo-black.png" alt=""> </a> </div>
      </div>

 

Icons

There are two kinds of font icons used in this template.

  1. Icofonts
  2. ET Icons

Icofonts is a @font-face iconset that allows you to change size and color of the icons just using CSS.

To add an icon, import Icofont from react-icofont pack, then simply create an Icofont element with attribute icon='[icon name]', for example:  

import Icofont from "react-icofont";

<Icofont icon="file" />
<Icofont icon="home" />
<Icofont icon="envelope" className=" whatever classes do you need " />

 

ET Icons is also a @font-face iconset that allows you to change size and color of the icons just using CSS.

To add an icon, simply create an empty element with class 'mdi-[icon name]', for example: 

<i class="mdi-anchor"></i>
<span class="mdi-fingerprint"></span>

Hero Units

You can use all hero units (slider sections) at any pages. Just import the component from components/HeroSliders folder and pass the data props in JSON format like we did.
Not all sliders have a data prop. First we recommend to check the component for its props.
Example:
If you wish to make your homepage with html video slider your new component have look like this snippet code from below. 

// We recommend to create new pages in the page folder,
   also the data JSON files to be in data folder.

import React from "react";
import Loader from "./../../components/Loader/Loader";
import HeroVideo from "../../components/Hero/HeroVideo";
import dataHero from "../../data/Sliders/hero-video-data.json";

const NewPage = () => (
	<Loader>
		<HeroVideo data={dataHero} />
	<Loader />
)

JSON: You can edit our current files data or create a new file. Whatever choice you, it's important to keep the structure data in the JSON file. Make sure that the provided path is correct

HTML5 Video

The HTML5 <video> element specifies a standard way to embed a video in a web page. You can play a video in your webpage background or any section.  You can host video form root server or can add link form other server. 

For browser support you have to provide two format video(.webm/.ogg and .mp4)

The markup: 

<div class="homepage-hero-module">
    <div class="video-container">
      <div class="filter"></div>
      <video autoplay loop class="fillWidth">
        <source src="assets/images/video/video.mp4" type="video/mp4" />
        <source src="assets/images/video/video.webm" type="video/webm" />
      </video>
      <div class="poster hidden"> <img src="assets/images/video/video.jpg" alt="video-img"> </div>
    </div>
  </div>

Customize logo

Header logo

To change your header logo you need to open MainLogo from ".src/components/MainLogo" and change the paths from the top imports. There are links for each version, black or white. For the desired effect we recommend to change both paths.

The top section of MainLogo component 

import React from "react";
import logoWhite from "../assets/images/logo-white.png"; // change for white logo
import logoBlack from "../assets/images/logo-black.png"; // change for black logo
import Icofont from "react-icofont";

Footer logo

To change your footer logo you need to open FooterOne from ".src/components/Footer/FooterOne" and change the path from the top imports.

The top section of FooterOne component 

import React from "react";
import logoFooter from "../../assets/images/logo-footer.png"; // change for logo
